Pinecone Big Data Demo
Proyecto académico desarrollado para el curso Datos Masivos.

Objetivo
Demostrar el funcionamiento básico de una base de datos vectorial mediante Pinecone.

Tecnologías utilizadas
Pinecone
Python
Vector Search
Big Data
RAG
Archivos
pinecone_demo.py
requirements.txt
Ejecución
Instalar dependencias:

pip install -r requirements.txt

Configurar API Key:

export PINECONE_API_KEY=“su_api_key”

Ejecutar:

python pinecone_demo.py

Curso
Datos Masivos

Tema
Pinecone como base de datos vectorial para Big Data e Inteligencia Artificial
